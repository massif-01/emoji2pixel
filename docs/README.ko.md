# 🎨 Emoji2Pixel

강력한 웹 기반 변환기로 이모지와 이미지를 아름다운 픽셀 아트로 변환하세요. 멋진 애니메이션을 만들고, 모든 세부 사항을 맞춤 설정하고, 작품을 이미지 또는 GIF로 내보낼 수 있습니다.

![Emoji2Pixel Badge](https://img.shields.io/badge/Emoji2Pixel-v1.0-brightgreen) ![License](https://img.shields.io/badge/license-MIT-blue) ![Languages](https://img.shields.io/badge/languages-8-orange)

## ✨ 기능

### 🖼️ **다중 소스 입력**
- **이모지 지원**: 이모지를 직접 입력하거나 포괄적인 이모지 라이브러리에서 검색
- **이미지 가져오기**: 픽셀화를 위해 모든 이미지 파일 업로드
- **자동 맞춤 모드**: 캔버스에 완벽하게 맞도록 이모지 크기 자동 조정
- **검색 및 필터**: 카테고리 탐색이 가능한 빠른 이모지 검색

### 🎬 **애니메이션 시스템**
- **키프레임 애니메이션**: 여러 키프레임으로 부드러운 애니메이션 생성
- **트윈 보간**: 키프레임 간 전환 프레임 자동 생성
- **재생 제어**: 실시간으로 재생, 일시 정지 및 애니메이션 속도 조정
- **GIF 내보내기**: 애니메이션을 애니메이션 GIF 파일로 내보내기

### 🎛️ **고급 변환 제어**
- **크기 조절**: 0%에서 200%까지 작품 크기 조정
- **위치**: 완벽한 정렬을 위한 X/Y 오프셋 미세 조정
- **회전**: 모든 방향으로 회전 (0-360°)
- **인터랙티브 캔버스**: 클릭 및 드래그로 직접 조작 + Shift를 누른 채 회전

### 🎨 **강력한 렌더링 엔진**
- **이중 렌더링 모드**:
  - **이상적 모드**: 맞춤형 간격과 테두리가 있는 전문 픽셀 아트
  - **베어 모드**: 진정한 레트로 스타일을 위한 순수 픽셀 렌더링
- **유연한 캔버스**: 조정 가능한 그리드 크기 (8x8에서 128x128 픽셀)
- **픽셀 스타일**: 정사각형, 둥근 모서리 또는 원형 픽셀 중 선택
- **색상 양자화**: 레트로 미학을 위해 2-256색으로 색상 팔레트 축소
- **선명도 필터**: 조정 가능한 강도로 가장자리 선명도 향상

### 🖌️ **편집 도구**
- **배경 제거**: 허용 오차 제어가 있는 지능형 배경 제거
- **선택 도구**: 채우기, 지우기, 복사 및 붙여넣기가 가능한 직사각형 선택
- **레이어 시스템**: 비파괴 편집 워크플로우
- **실행 취소 지원**: 색상 선택 및 배경 제거 작업 되돌리기

### 📏 **전문가용 내보내기 옵션**
- **다중 단위**: 밀리미터, 인치 또는 그리드 단위로 작업
- **크기 프리셋**: 일반적인 디스플레이 크기에 대한 빠른 프리셋
- **내보내기 형식**:
  - PNG (투명도 포함)
  - GIF (애니메이션 또는 정적)
  - 원시 픽셀 데이터
- **프레임 렌더링**: 실제 크기로 물리적 픽셀 레이아웃 시각화

### 로컬 개발

1. **저장소 복제**
   ```bash
   git clone https://github.com/thomas-hiddenpeak/emoji2pixel.git
   cd emoji2pixel
   ```

2. **로컬 서빙**
   
   Python 사용:
   ```bash
   python -m http.server 8000
   ```
   
   또는 Node.js 사용:
   ```bash
   npx http-server -p 8000
   ```

3. **브라우저에서 열기**
   ```
   http://localhost:8000
   ```

빌드 프로세스가 필요 없습니다! 순수 정적 웹 애플리케이션입니다.

## 📖 사용 가이드

### 기본 워크플로우

1. **입력**: 이모지를 입력하거나 이미지 업로드
2. **변환**: 원하는 대로 크기, 위치 및 회전 조정
3. **프레임 추가**: `+` 버튼을 클릭하여 애니메이션에 추가
4. **사용자 정의**: 렌더링 설정, 픽셀 스타일 및 색상 조정
5. **내보내기**: PNG 또는 GIF로 다운로드

### 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| `스페이스` | 애니메이션 재생 전환 |
| `Enter` | 현재 보기를 키프레임으로 추가 |
| `Delete` / `Backspace` | 선택한 프레임 삭제 |
| `←` / `→` | 프레임 간 탐색 |
| `Esc` | 선택/색상 선택 취소 |
| `Ctrl/Cmd + C` | 선택 영역 복사 |
| `Ctrl/Cmd + V` | 선택 영역 붙여넣기 |

### 프로 팁

- 🎯 캔버스에서 드래그하는 동안 **Shift를 누른 채**로 이동이 아닌 회전
- 🔍 진정한 레트로 픽셀 아트를 위해 **색상 양자화** (8-64색) 사용
- ⚡ 가장자리 선명도를 향상시키기 위해 **선명도** (30-50% 강도) 활성화
- 🎬 부드러운 애니메이션을 위해 **트윈 프레임**을 5-10으로 설정
- 📐 물리적 LED 매트릭스 레이아웃을 시각화하기 위해 **프레임 렌더링 모드** 사용

## 🛠️ 기술 스택

- **순수 프론트엔드**: HTML5, CSS3, 바닐라 JavaScript
- **종속성 없음**: 외부 라이브러리 또는 프레임워크 제로
- **Canvas API**: 고성능 픽셀 조작
- **GIF.js**: 클라이언트 측 GIF 인코딩
- **반응형 디자인**: 데스크톱 및 태블릿 장치에서 작동

## 📁 프로젝트 구조

```
emoji2pixel/
├── index.html          # 메인 HTML 구조
├── app.js              # 핵심 애플리케이션 로직
├── styles.css          # 스타일 및 레이아웃
├── locales/            # 국제화
│   ├── index.json      # 로케일 매니페스트
│   ├── ko-KR.json      # 한국어 번역
│   ├── en-US.json      # 영어 번역
│   └── ...             # 기타 언어
├── docs/               # 다국어 문서
│   ├── README.ko.md    # 한국어 문서
│   ├── README.en.md    # 영어 문서
│   └── ...             # 기타 언어
└── scripts/            # 빌드 유틸리티
    └── generate_locales_index.py
```

## 🌐 새 언어 추가

1. `locales/`에 새 로케일 파일 생성 (예: `pt-BR.json`)
2. 기존 로케일 파일에서 구조 복사
3. 대상 언어로 모든 키 번역
4. 국기 이모지가 있는 `selfName` 필드 추가
5. 로케일 인덱스 생성기 실행:
   ```bash
   python scripts/generate_locales_index.py
   ```

새 언어가 언어 선택기에 자동으로 나타납니다!
