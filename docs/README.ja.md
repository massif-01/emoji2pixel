# 🎨 Emoji2Pixel

この強力なWebベースのコンバーターで、絵文字や画像を美しいピクセルアートに変換しましょう。素晴らしいアニメーションを作成し、細部までカスタマイズし、作品を画像やGIFとしてエクスポートできます。

**コア機能**：WS2812 LEDマトリックスディスプレイ用のプロフェッショナルなアニメーションとピクセルアートプレビュー、リアルなLEDレンダリングシミュレーション付き。

## ✨ 機能

### 🖼️ **マルチソース入力**
- **絵文字サポート**：絵文字を直接入力するか、包括的な絵文字ライブラリから選択
- **画像インポート**：任意の画像ファイルをアップロードしてピクセル化
- **自動フィットモード**：絵文字のスケールを自動調整してキャンバスに完璧にフィット
- **検索とフィルター**：カテゴリナビゲーション付きの高速絵文字検索

### 🎬 **アニメーションシステム**
- **キーフレームアニメーション**：複数のキーフレームでスムーズなアニメーションを作成
- **トゥイーン補間**：キーフレーム間の遷移フレームを自動生成
- **再生コントロール**：リアルタイムで再生、一時停止、アニメーション速度を調整
- **GIFエクスポート**：アニメーションをアニメーションGIFファイルとしてエクスポート

### 🎛️ **高度な変換コントロール**
- **スケール**：0%から200%まで作品のサイズを変更
- **位置**：完璧な配置のためにX/Yオフセットを微調整
- **回転**：任意の方向に回転（0-360°）
- **インタラクティブキャンバス**：クリック&ドラッグで直接操作 + Shiftを押しながら回転

### 🎨 **強力なレンダリングエンジン（WS2812 LEDマトリックス プロフェッショナルプレビュー）**
- **2つのレンダリングモード**：
  - **理想モード**：ディフューザー付きWS2812をシミュレート - 各LEDの光が均等に拡散してプロフェッショナルな表示効果を実現
  - **ベアモード**：裸のWS2812の外観をシミュレート - 拡散なしの直接LEDピクセルポイント、生のLED効果を表示
- **柔軟なキャンバス**：調整可能なグリッドサイズ（8x8から128x128ピクセル）
- **ピクセルスタイル**：四角、角丸、円形のピクセルから選択
- **色の量子化**：2〜256色にカラーパレットを削減してレトロな美学を実現
- **シャープニングフィルター**：調整可能な強度でエッジの鮮明度を向上

### 🖌️ **編集ツール**
- **背景除去**：許容範囲コントロール付きのインテリジェントな背景除去
- **選択ツール**：塗りつぶし、消去、コピー＆ペースト付きの矩形選択
- **レイヤーシステム**：非破壊的な編集ワークフロー
- **元に戻すサポート**：カラーピッキングと背景除去操作を元に戻す

### 📏 **プロフェッショナルなエクスポートオプション**
- **複数の単位**：ミリメートル、インチ、またはグリッド単位で作業
- **サイズプリセット**：一般的なディスプレイサイズのクイックプリセット
- **エクスポート形式**：
  - PNG（透明度付き）
  - GIF（アニメーションまたは静止画）
  - 生のピクセルデータ
- **フレームレンダリング**：実世界の寸法で物理的なピクセルレイアウトを視覚化

### ローカル開発

1. **リポジトリをクローン**
   ```bash
   git clone https://github.com/thomas-hiddenpeak/emoji2pixel.git
   cd emoji2pixel
   ```

2. **ローカルで提供**
   
   Pythonを使用：
   ```bash
   python -m http.server 8000
   ```
   
   またはNode.jsを使用：
   ```bash
   npx http-server -p 8000
   ```

3. **ブラウザで開く**
   ```
   http://localhost:8000
   ```

ビルドプロセスは不要です！これは純粋な静的Webアプリケーションです。

## 📖 使用ガイド

### 基本的なワークフロー

1. **入力**：絵文字を入力するか画像をアップロード
2. **変換**：お好みに応じてスケール、位置、回転を調整
3. **フレームを追加**：`+`ボタンをクリックしてアニメーションに追加
4. **カスタマイズ**：レンダリング設定、ピクセルスタイル、色を調整
5. **エクスポート**：PNGまたはGIFとしてダウンロード

### キーボードショートカット

| ショートカット | アクション |
|---------------|----------|
| `スペース` | アニメーション再生の切り替え |
| `Enter` | 現在のビューをキーフレームとして追加 |
| `Delete` / `Backspace` | 選択したフレームを削除 |
| `←` / `→` | フレーム間を移動 |
| `Esc` | 選択/カラーピッキングをキャンセル |
| `Ctrl/Cmd + C` | 選択範囲をコピー |
| `Ctrl/Cmd + V` | 選択範囲を貼り付け |

### プロのヒント

- 🎯 キャンバス上でドラッグ中に**Shiftを押したまま**にすると、移動ではなく回転します
- 🔍 本格的なレトロピクセルアートには**色の量子化**（8〜64色）を使用
- ⚡ エッジの明瞭度を向上させるために**シャープニング**（30〜50%の強度）を有効化
- 🎬 スムーズなアニメーションのために**トゥイーンフレーム**を5〜10に設定
- 📐 物理的なLEDマトリックスレイアウトを視覚化するために**フレームレンダリングモード**を使用

## 🛠️ 技術スタック

## 🌐 新しい言語の追加

1. `locales/`に新しいロケールファイルを作成（例：`pt-BR.json`）
2. 既存のロケールファイルから構造をコピー
3. すべてのキーをターゲット言語に翻訳
4. フラグ絵文字付きの`selfName`フィールドを追加
5. ロケールインデックスジェネレーターを実行：
   ```bash
   python scripts/generate_locales_index.py
   ```

新しい言語は言語セレクターに自動的に表示されます！
